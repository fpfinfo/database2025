PWA Package - Simulador de Recomposição Salarial

Antes de usar:
1. Crie uma pasta 'icons' na raiz do projeto.
2. Adicione os arquivos de ícone:
   - icons/icon-192.png
   - icons/icon-512.png
   (tamanhos exatos conforme manifest.json)

Para testar:
- Sirva via HTTPS (ou 'localhost').
- Acesse 'index.html' no navegador.
- Aceite o prompt de instalação para adicionar como app.

Pronto!
